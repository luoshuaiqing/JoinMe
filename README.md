# JoinMe
This is an Employment Marketplace for Job Seekers and Employers that supports direct chatting between the two.
## Key functionalities: 
* chatting
* email verification
* autocomplete
* login/signup
* keyword searching
* page transition effect 
## Key technologies:
* SCSS
* NPM
* Session
* PHP
* jQuery
* jQuery UI
* Media Queries
* JavaScript
* CSS
* HTML5
